# Synch Project Roadmap

## 🎯 v0.1.0 - MVP Release (Current)
- [x] Rust Core with Ed25519/X25519 Encryption.
- [x] Basic Vault Delta synchronization logic.
- [x] Go Relay Server with WebSocket support.
- [x] Preliminary Android & VCP-Agent integration.
- [x] CI/CD Pipeline for automated testing and releases.

## 🚀 v0.2.0 - Federation & Resilience
- [ ] **Federation Protocol**: Support for multiple inter-connected relay servers.
- [ ] **Persistence Layer**: Pluggable storage backends (PostgreSQL, SQLite, S3).
- [ ] **Enhanced Conflict Resolution**: Better handling of concurrent delta updates.
- [ ] **Desktop Client**: Full-featured Rust/Tauri desktop application.

## 🛡️ v0.3.0 - Privacy & Governance
- [ ] **Zero-Knowledge Proofs**: For identity verification without revealing public keys.
- [ ] **Decentralized Identifiers (DID)**: Integration with W3C DID standards.
- [ ] **Governance Mechanism**: On-chain or off-chain voting for protocol upgrades.

## 🌐 v1.0.0 - Production Readiness
- [ ] **Audit**: Third-party security and cryptography audit.
- [ ] **Ecosystem**: Rich plugin marketplace for Agent platforms.
- [ ] **Global Relay Network**: High availability across multiple cloud providers.
